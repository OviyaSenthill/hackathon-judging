# Hackathon Judging Dashboard

Internal tool for managing project scores and track winners.

## How to Use

1. Open the dashboard: https://OviyaSenthill.github.io/hackathon-judging/
2. Import teams from the Devpost CSV export
3. Enter judge scores manually from the Google Form responses
4. View leaderboard and track winners
5. Export final results as CSV

## Day-of Workflow

- One designated person runs the dashboard as the scorekeeper
- Judges submit scores via Google Form
- Scorekeeper enters scores into the dashboard as they come in
- After judging closes, export CSV for final results

## Files

- `index.html` — the dashboard app
- `results/` — exported result CSVs after the event# hackathon-judging
